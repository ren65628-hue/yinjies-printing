# 🚀 印界印刷官网 - 2 分钟上线指南

## 方案：Netlify Drop（最简单！无需注册！）

---

## 📋 操作步骤（只需 3 步！）

### 第 1 步：打开网站
在浏览器中打开：
```
https://app.netlify.com/drop
```

### 第 2 步：拖拽上传
找到电脑上的这个文件夹：
```
/home/admin/.openclaw/workspace/website
```
把整个 `website` 文件夹**拖到**Netlify Drop 页面的虚线框里

### 第 3 步：完成！
等待几秒钟，网站就上线了！

你会得到一个免费网址，类似：
```
https://yinjies-printing.netlify.app
```
