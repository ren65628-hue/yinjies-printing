# 🚀 印界印刷官网 - 部署指南

## 方案一：Netlify Drop（推荐 ⭐⭐⭐⭐⭐）

**最简单！无需注册！2 分钟上线！**

### 步骤：

1. **打开网站**
   - 访问：https://app.netlify.com/drop

2. **上传网站文件夹**
   - 把 `website` 文件夹拖到页面上
   - 或者点击选择文件夹

3. **完成！**
   - 网站立即上线
   - 获得免费域名：`https://xxx-xxx.netlify.app`

4. **可选：注册账号绑定**
   - 注册免费账号后绑定网站
   - 可以绑定自定义域名

---

## 方案二：GitHub Pages（永久免费）

**需要 GitHub 账号，但更专业**

### 步骤：

1. **注册 GitHub**
   - 访问：https://github.com/signup
   - 免费注册账号

2. **创建仓库**
   - 仓库名：`yinjies-printing` 或 `yinjies-printing.github.io`
   - 设为公开 (Public)

3. **上传文件**
   - 把 `index.html` 上传到仓库

4. **启用 Pages**
   - Settings → Pages
   - Source 选择 `main branch`
   - 保存

5. **完成！**
   - 网站地址：`https://用户名.github.io/yinjies-printing`

---

## 方案三：Vercel（推荐 ⭐⭐⭐⭐⭐）

**快速、免费、专业**

### 步骤：

1. **访问** https://vercel.com/new

2. **导入项目**
   - 可以用 GitHub 导入
   - 或者直接拖拽上传

3. **完成！**
   - 获得域名：`https://xxx.vercel.app`
